# 🌠 StarGuars

## ¡Esquiva los meteoritos y alcanza la cima del podio!

StarGuars es un trepidante juego de acción inspirado en *Space Invaders*, donde deberás esquivar meteoritos y acumular la mayor cantidad de puntos posible. ¡Compite por el mejor puntaje y demuestra tu habilidad en el espacio!

## 🚀 Características  
- 🎮 Jugabilidad sencilla pero desafiante.  
- ☄️ Diferentes tipos de meteoritos con patrones únicos.  
- 🏆 Sistema de puntuación y clasificación.  
- 🔊 Efectos de sonido y gráficos retro espaciales.  

## 🎯 ¿Cómo jugar?
1. Mueve tu nave con las **flechas del teclado** o **[insertar controles]**.  
2. Esquiva los meteoritos que caen a toda velocidad.  
3. Acumula puntos por cada segundo que sobrevivas.  
4. ¡Llega lo más lejos posible y sube al podio!
